# AI 툴·모델 리포스트 (2026-01-29)

## Hugging Face Spaces: 가입만 하면 무료로 써볼 수 있는 AI 서비스
- 이미지 생성, 음성 인식, 번역, 챗봇 등 설치 없이 웹에서 바로 사용 가능
- 요지: 생각보다 많은 사람들이 Hugging Face Spaces에 공개된 수만 개의 데모/서비스를 놓치고 있음

## Tencent: HY-Motion 1.0 (텍스트 → 3D 캐릭터 모션)
- 텍스트 지시만으로 고품질 3D 캐릭터 동작 생성
- DiT(Diffusion Transformer) + Flow Matching 기반
- 10억(1B) 파라미터 규모
- 특징: SFT를 넘어 RLHF까지 적용해 물리적 어색함 감소
- 예: “검을 휘두르며 걷기” 같은 복합 지시도 자연스럽게 수행
- 200개+ 동작 카테고리 커버 → 게임 NPC/애니메이션 파이프라인에 즉시 활용 가능

## Microsoft: TRELLIS 2 (이미지 1장 → 3D 모델)
- 40억(4B) 파라미터
- “O-Voxel” 구조로 복잡한 형태(옷, 나뭇잎 등)도 고퀄리티 3D로 생성
- 속도:
  - 512 해상도 기준 ~3초
  - 1536 해상도도 ~1분
- 텍스처/PBR 속성(roughness/metallic/opacity 등) 보존
- MIT License

## Alibaba: Z-Image / Z-Image Turbo (오픈소스 이미지 생성)
- 6B 오픈소스 이미지 생성 모델
- Turbo: 1초 이내 생성 속도 + 프롬프트 해석력 호평
- 비용 언급: 1MP 당 $0.005 수준
- Flux2, Qwen-Image 등과 비교에서 경쟁력 언급

## Resemble AI: Chatterbox Turbo (오픈소스 음성 모델)
- 벤치마크에서 ElevenLabs/Cartesia 대비 우수하다는 평가 언급
- 5초 오디오로 음성 복제
- laugh/sigh 등 비언어 태그 표현 가능
- 속도: 6배 빠름 → 실시간 대화 적합
- MIT License + PerTh 워터마킹 탑재(딥페이크 악용 우려 완화 목적)

## OmniPart: 3D 오브젝트를 ‘부품 단위’로 생성/제어
- 3D 오브젝트를 part 단위로 분해하고
- 구조를 계획한 뒤
- 동시에 생성하여 일관된 3D 형상 출력

## Qwen Image Edit + LoRA 흐름
- Qwen Image Edit에 LoRA를 붙이는 흐름이 커지면서
- Hugging Face에 편집용 LoRA가 다수 공유되는 중
- 스타일 복원, 얼굴 톤 유지 같은 기능을 별도 학습한 어댑터들
- “몇백 장” 수준의 데이터로 학습 가능 → 가능성 확대
