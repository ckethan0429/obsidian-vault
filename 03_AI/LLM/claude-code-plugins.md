# Claude Code Plugins

The official Anthropic-managed plugin directory surfaced on GitHub Trending on 2026-05-19, which makes plugins a first-class distribution surface for Claude Code alongside skills and subagents.

## Why this matters

- Plugins are not just individual repos; they are a packaging and distribution layer.
- The official directory makes the plugin surface easier to reason about, curate, and route into durable notes.
- This keeps plugins in the same practical family as subagents, skills, hooks, and managed agent workflows.
