# Interest Signal Sources

This page records the main inputs CK wants the wiki to watch when building an interest map.
It is the canonical wiki version of the earlier Obsidian note `관심사 소스 목록`.

## Source list
- YouTube subscriptions: only newly published videos, excluding sports and entertainment-heavy items, and only the parts relevant to CK's actual interests
- X: likes and reposts
- Threads: likes and reposts
- GitHub: stars on repositories, especially Claude Code / agent tooling / infrastructure repos
- Google search history
- ChatGPT: answers to CK's questions
- Claude: answers to CK's questions

## Filtering rule
- Treat repeated topics across these sources as stronger interest signals
- Ignore low-signal sports/entertainment items from YouTube
- Prefer concrete topics, questions, and recurring themes over generic browsing

## How to use this page
- Feed these sources into future wiki ingests
- Use them to find recurring domains, product ideas, and research topics
- Link to pages created from the extracted signals
