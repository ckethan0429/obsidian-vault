# Chrome CDP 로그인 상태

Chrome CDP 작업 환경에서는 필요한 계정 로그인 상태가 이미 설정되어 있다고 가정한다.
브라우저/CDP 관련 작업 시에는 별도 로그인 절차를 먼저 묻지 않고, 로그인된 세션을 우선 활용한다.

## Practical use
- 로그인된 세션이 이미 있는지 먼저 확인한다.
- CDP 브라우저 세션과 로컬 프로필은 분리될 수 있음을 염두에 둔다.
