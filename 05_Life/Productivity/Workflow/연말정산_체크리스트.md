# 연말정산 체크리스트 (리포스트 정리)

> 📌 자동으로 안 챙겨주는 항목이 많아서, 모르면 0원 처리될 수 있음.

## 체크 포인트
1. 헬스·수영장 → 30% 공제 (추가 신설)
2. 월세 → 제출 서류 필수, 집주인 동의 필요 X
3. 안경/렌즈 구입비 → 자동 반영 안 될 수 있음
4. 미취학 아동 교육비 → 자동 반영 안 될 수 있음
5. 혼인신고 → 부부 합산 100만 원
6. 산후조리비 → 최대 200만
7. 난임시술비 → 30%까지
