# gstack

`gstack` is Garry Tan's open-source skill and workflow system for AI-assisted product development. It is positioned as a software factory that turns an AI coding agent into a virtual engineering team with specialized roles across planning, review, QA, security, release, and retrospectives.

## What the source emphasizes
- Founder- and CEO-friendly workflow for shipping with AI agents
- Slash-command-based specialist roles such as CEO review, engineering review, design review, QA, security, release, and retrospectives
- Strong process framing: think → plan → build → review → test → ship → reflect
- Support for Claude Code first, with paths for OpenClaw and other coding agents

## Why it matters
The repo is relevant as a practical example of AI-native productivity infrastructure for startup builders. It translates abstract agentic coding ideas into repeatable operating rituals and review pipelines.

## Connection to this wiki
`gstack` sits at the intersection of startup execution, AI-agent operations, and product development methodology. It is especially useful as a reference point for how solo or small-team builders can scale output with agent workflows.
